# Calculator Application – Java Swing

A GUI-based Calculator application developed using Java Swing and event-driven programming.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Clean and user-friendly GUI
- Event-driven button handling
- Error-free execution

## Technologies Used
- Java
- Java Swing
- AWT
- Event Handling

## How to Run
1. Clone the repository
2. Compile the Calculator.java file
3. Run the program
